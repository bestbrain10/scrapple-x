## How to Install

Install NodeJS and NPM. Use this link and follow the steps for your own operating system https://nodejs.org/en/download/

Install Git from https://git-scm.com/downloads

From your terminal or command line run

```bash
# clone from githhub
git clone git@github.com:bestbrain10/scrapple-x.git

# move into project directory
cd scrapple-x

# install dependencies using NPM
npm install
```


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
